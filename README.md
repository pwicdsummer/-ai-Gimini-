# -ai-Gemini-
本项目最终目标在于实现从常用聊天ai获取用户与AI的聊天记录（存储在本地）、基于数据生成点线状思维导图以实现辅助学习的作用。作者已经试着做过一个demo验证了项目思路的可行性，但由于个人开发水平与精力的限制目前还无法制作出一个一体化的作品。该项目会持续更新，欢迎有需求的各位下载体验！

运行环境：
python

项目文件：

插件包

Content Saving.py

flask_install.bat

*******使用指南*******

1.下载项目文件夹并解压

2.在任意浏览器中找到“插件”，启用“开发者模式”，添加新插件（即项目文件中包括的名为“插件包”的文件夹），刷新

3.双击“flask_install.bat”，安装或检测flask是否存在

4.点击Content Saving.py，以“python”方式运行

然后就可以去找Gemini测试了o(*￣▽￣*)ブ
